源码说明：
1、cosinSimilarity.py:主要功能实现了两个向量之间的cosin值计算
2、urlLib.py:主要功能是通过combine、set1、set2给定的词汇，通过google爬取相关页面，获得每个词出现的次数，以及共同词对出现的次数，运用基于检索页面数量的方法计算相似度
3、Similarity.py:主要功能实现了wordnet自带的6种词汇相似度计算方法，以及运用google给出的英文语料，训练出了词向量模型modelHuang,并且给出了sperman差异度的计算方法。
4、SVM.py：主要实现了SVM训练回归模型的功能
5、spearman.py：主要用于计算spearman系数
